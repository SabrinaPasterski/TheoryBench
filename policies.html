TheoryBench ✦beta About Launch

Before you download

TheoryBench runs on your Mac. You choose the folder, the accounts, and what leaves your machine.

Control what has access

Choose a paper folder

Open the TeX, figures, bibliography, and notes you already use.

Keep working notes local

Review history and notebook outputs stay local unless exported.

Keep Overleaf in sync

Pull and push paper files while Overleaf remains the shared editor.

Choose when AI can read

AI review uses the local project index to send relevant draft context to Claude or Codex. Turn it off to keep drafts local.

Requirements

TeX

Install TeX Live or MacTeX to compile PDFs locally. The app shows which TeX and bibliography tools it finds.

Wolfram

Install Mathematica or Wolfram Engine to run Wolfram cells with your own local license.

Python (optional)

Use Python cells when you want them. Missing packages are reported, and outputs stay with the paper.

Claude/Codex (optional)

Connect Claude or Codex for Theoria review. A local project index supplies relevant draft context to the account you choose.

Beta limits

Overleaf conflicts

If Overleaf and local edits conflict, the app stops before pushing.

Local setup matters

PDFs and computations depend on the tools installed on your Mac.

Early access: the demo is live. The first downloadable app is for macOS and is still being packaged and tested.
